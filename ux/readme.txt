For the development of the skinning/User Interface. The User Interface should allow the scenarios outlined in the Staker and Borrowing documents. 

This Gui makes use of Qt5 in python, please download anaconda or find the most up to date qt5 development packages for python to run this prototype.

The Seed Generate Makes use of a dictionary of 100's of common words and creates a 16 word seed for the prototype

Generating Public Private Key makes use of cryptographic math, we use modular arithmetic, detect large prime numbers and creates an RSA cypher to generate public and private keys
